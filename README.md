# 臺灣民俗體育研究資料整理專案

本 repository 旨在建立「臺灣民俗體育研究資料」之整理、分類、筆記與版本管理架構，作為後續文獻回顧、田野觀察、研究計畫撰寫、論文章節發展與教學資料建置之基礎。

本專案特別關注臺灣民俗體育、陣頭活動、客家獅、宋江陣、金獅陣、舞龍舞獅、廟會身體文化、族群認同、文化再現、後殖民影響與全球化脈絡下的傳統身體文化變遷。

## 一、專案目的

1. 建立民俗體育研究資料之系統性分類架構。
2. 整理中文、外文期刊與歷史文獻之閱讀筆記。
3. 建構後殖民理論、文化霸權、文化再現、族群認同與全球化等理論筆記。
4. 保存研究計畫、論文草稿、田野觀察格式與資料整理流程。
5. 作為未來研究助理、學生或合作研究者協作之版本管理平台。

## 二、資料分類架構

本專案初步分為以下幾個部分：

```text
01_研究計畫與主題/
02_文獻資料整理/
03_民俗體育主題分類/
04_理論架構/
05_田野觀察與訪談/
06_計畫書與論文草稿/
99_待辦事項/
```

## 三、使用原則

1. 本 repository 主要存放研究架構、文字筆記、文獻清單、Markdown 草稿與資料索引。
2. 大型 PDF、Word、照片、錄音、影片與掃描檔，建議另存於 Google Drive、外接硬碟或其他安全雲端空間。
3. 涉及受訪者、學生、合作單位、電話、地址、簽名、未公開資料或其他個人資料者，不建議直接放入公開 GitHub。
4. 所有研究筆記應盡量標示來源、日期、關鍵字與後續可應用之研究章節。
5. 文獻引用資料宜逐步整理為 APA 第七版格式。

## 四、後續工作方向

1. 補齊中文文獻與外文文獻清單。
2. 建立客家獅、宋江陣、金獅陣、舞龍舞獅等主題筆記。
3. 整理後殖民、文化霸權、文化再現、族群認同與全球化相關理論。
4. 建立田野觀察與訪談紀錄格式。
5. 逐步形成國科會計畫、期刊論文與博士級研究論述之資料基礎。
