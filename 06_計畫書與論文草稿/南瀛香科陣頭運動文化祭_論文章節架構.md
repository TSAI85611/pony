## 二、計畫書轉化為論文之核心問題意識

本計畫原以「南瀛香科陣頭運動文化祭」作為地方特色全民運動活動申辦案，強調以臺南香科、宮廟、庄頭陣頭、學校及社區場域為基礎，將宋江陣、金獅陣、舞龍舞獅、跳鼓、車鼓、牛犂陣、高蹺陣等傳統身體文化，轉化為全民可參與、可學習、可體驗、可保存與可品牌經營之地方運動資源。

若轉化為論文，其問題意識可集中於以下四點：

1. **民俗體育如何被重新定義為全民運動？**  
   傳統陣頭不僅是宗教展演或民俗活動，也包含步法、身段、節奏、隊形、器械操作、協力默契、體能訓練與集體紀律等運動元素。

2. **地方香科場域如何成為運動品牌的文化基礎？**  
   臺南香科、宮廟、庄頭與學校場域不只是活動地點，更是地方記憶、信仰秩序、社群連結與身體文化實踐的場域。

3. **分散式、分期、分區、分眾的文化祭模式如何降低大型活動風險？**  
   本計畫避免一次性大型動員，改採「香科場域＋陣頭申請＋研習體驗＋廟會展演＋影像保存＋品牌整合」架構，兼顧文化真實性、經費合理性與安全管理。

4. **民俗體育如何透過政策語言形成城市運動品牌？**  
   本案以「南瀛香科陣頭運動文化祭」為品牌名稱，以「陣動南瀛，全民入陣」為標語，將地方陣頭從被觀看的民俗展演，轉化為可實作、可參與、可保存的全民運動文化。

## 三、論文章節總架構

## 第一章　緒論

### 1.1 研究背景

臺南長期具有深厚的香科、宮廟、陣頭與民俗體育資源。香科場域中的宋江陣、金獅陣、舞龍舞獅、跳鼓、車鼓、牛犂陣、高蹺陣等身體文化，兼具儀式性、社群性、技藝性與運動性。傳統上，這些活動常被視為宗教展演或地方民俗，但從體育社會學觀點觀察，其內部包含訓練、協作、節奏、身體控制、隊形移動與集體紀律，具備轉化為全民運動的理論與實務基礎。

### 1.2 研究動機

115年地方特色全民運動活動申辦計畫強調「運動＋在地」與地方運動品牌建構，而「南瀛香科陣頭運動文化祭」正是以臺南香科場域與陣頭文化為基礎，試圖將傳統民俗體育轉譯為具故事性、參與感、辨識度與延續性的公共運動事件。此一案例有助於探討地方文化如何透過政策設計、活動策展與身體實踐，形成新的全民運動治理模式。

### 1.3 研究目的

1. 分析臺南香科與陣頭文化作為地方特色全民運動品牌之基礎。
2. 探討傳統陣頭如何由民俗展演轉化為可參與、可體驗、可教學之運動形式。
3. 檢視「香科場域型陣頭運動文化祭」之執行模式、品牌策略與風險治理。
4. 建構民俗體育轉化為全民運動政策實踐之分析架構。

### 1.4 研究問題

1. 臺南陣頭文化中哪些身體技藝可被轉譯為全民運動資源？
2. 「南瀛香科陣頭運動文化祭」如何結合香科、宮廟、社區、學校與地方品牌？
3. 分散式文化祭模式如何兼顧文化真實性、全民參與、經費合理性與安全控管？
4. 此計畫如何回應地方特色全民運動活動申辦計畫之審查邏輯與政策目標？

### 1.5 研究範圍與限制

本研究以「南瀛香科陣頭運動文化祭」計畫書為主要文本，並以臺南市五大香科相關區域、代表性宮廟廣場、庄頭社區、學校場域與期末成果發表場地為分析範圍。研究限制在於：計畫目前屬申辦與規劃階段，若尚未完成實際執行，成效分析須先以計畫設計、預期效益、KPI與風險治理機制作為分析基礎，後續再補充田野觀察與成果資料。

## 第二章　政策脈絡與計畫依據

### 2.1 地方特色全民運動活動申辦計畫之政策精神

本節說明運動部全民運動署地方特色全民運動活動申辦計畫之政策方向，特別是：

1. 從一般補助活動轉向地方運動品牌經營。
2. 以「運動＋在地」為核心。
3. 結合文化節慶、季節時令、產業元素與特定族群需求。
4. 將運動轉化為具故事性、參與感與公共性的地方事件。
5. 推動可累積、可辨識、可傳承的全民運動文化。

### 2.2 審查指標與論文分析面向

地方特色全民運動活動申辦計畫之審查重點可轉化為本研究的分析面向：

| 申辦審查重點 | 論文分析面向 |
|---|---|
| 活動具特色 | 香科場域、宮廟文化、陣頭身體技藝、地方辨識性 |
| 節日及族群特色 | 香科週期、親子、青少年、社區、銀髮與一般民眾參與 |
| 活動可行性／完整性／創新性 | 分區、分期、分眾、申請制與精華展演模式 |
| 主題運動推廣課程 | 12週、30小時以上之陣頭運動體驗課程設計 |
| 品牌經營 | 主視覺、品牌標語、社群標籤、影音資料庫與成果手冊 |
| 宣傳推廣與整合行銷 | 社群短影音、紀錄影片、地方合作與城市行銷 |
| 經費合理性 | 總經費、自籌比率、分散式活動降低大型動員成本 |
| 風險管理與KPI | 保險、滿意度調查、參與人數、執行率、成果填報 |

### 2.3 申辦計畫書與學術研究之轉換邏輯

本節可說明：申辦計畫書重在政策符合性、活動可行性與經費合理性；論文則需進一步分析其背後的文化邏輯、身體實踐、地方治理、政策轉譯與社會效益。因此，本研究不是單純描述活動，而是將活動視為民俗體育政策化、品牌化與全民化的案例。

## 第四章　理論架構

### 4.1 民俗體育的運動化轉譯

本研究可將「陣頭運動」視為傳統民俗活動向全民運動轉譯的過程。轉譯不是簡化傳統，而是在保留文化脈絡的前提下，將其身體技藝、節奏、隊形、協作與訓練元素轉化為一般民眾可參與的運動形式。

### 4.2 文化再現與地方認同

「南瀛香科陣頭運動文化祭」不只是活動命名，而是將臺南地方文化重新編碼為城市運動品牌。可分析品牌名稱、標語「陣動南瀛，全民入陣」、視覺意象與社群標籤如何形成地方文化再現。

### 4.3 後殖民與在地身體知識

傳統陣頭長期位於現代體育制度之外，常被歸為民俗、宗教或表演。本研究可從後殖民與在地知識角度，討論陣頭身體文化如何被重新承認為具有運動價值、教育價值與政策價值的地方知識。

### 4.4 文化治理與跨域合作

本計畫整合學校、武館、宮廟、社區、影像團隊、地方政府與企業資源，形成地方運動治理網絡。此部分可使用文化治理、協力治理與跨域合作理論進行分析。

1. 115年地方特色全民運動活動申辦計畫相關規範。
2. 「南瀛香科陣頭運動文化祭」計畫書草案。
3. 臺南香科、宮廟、陣頭與民俗體育相關文獻。

5. 後續可補充訪談資料、田野觀察紀錄與參與觀察資料。

### 5.3 分析方法

1. **政策文本分析**：分析申辦計畫之政策目標、審查指標與KPI。
2. **計畫文本分析**：分析南瀛香科陣頭運動文化祭之品牌設計、執行模組與風險治理。
3. **田野觀察**：觀察課程、工作坊、廟會場域展演與期末成果發表。

## 第六章　個案分析：南瀛香科陣頭運動文化祭之品牌建構

### 6.1 活動基本資料

| 項目 | 內容 |
|---|---|
| 活動名稱 | 南瀛香科陣頭運動文化祭：臺南香科場域陣頭運動品牌推廣計畫 |
| 申請單位 | 國立臺南大學（草案） |
| 期程 | 核定日起至115年11月30日止 |

| 參與規模 | 活動參與人次至少3,500人；主題課程固定學員至少30人；累計課程參與人次360人以上 |

品牌核心為：「以香科為場域、以陣頭為身體、以運動為方法、以文化為故事、以影像為延續」。此定位將陣頭由單純表演或民俗展示，轉化為地方公共運動事件。

### 6.3 品牌符號系統

| 項目 | 內容 |
|---|---|
| 品牌名稱 | 南瀛香科陣頭運動文化祭 |
| 品牌標語 | 陣動南瀛，全民入陣 |

| 視覺意象 | 香科陣旗、鼓點節奏、龍獅動線、廟埕紅磚、隊形移動與人群共動 |
| 社群標籤 | #南瀛香科陣頭運動文化祭 #陣動南瀛 #全民入陣 #臺南陣頭運動 |

本研究可將陣頭運動分為三類：

1. **武藝型陣頭運動**：重視步法、身段、器械、隊形、肌力、敏捷、反應與團隊紀律。
2. **龍獅鼓樂型陣頭運動**：重視心肺耐力、節奏協調、核心肌群、協力默契與空間移動。
3. **民俗律動型陣頭運動**：重視肢體律動、平衡感、節奏感、地方故事、親子與社區參與。

## 第七章　執行模式分析：從活動計畫到地方運動治理

### 7.1 六大工作模組

本計畫採「香科場域＋陣頭申請＋研習體驗＋廟會展演＋影像保存＋品牌整合」六大工作模組：

1. **南瀛香科陣頭運動資源盤點**：盤點臺南五大香科、代表性宮廟、武館、文武陣頭、學校民俗體育社團、社區團隊與可合作場域。
2. **代表性陣頭運動團隊申請與輔導**：徵選10至12組具代表性與執行能力之團隊。
3. **主題運動推廣課程**：開設12週、30小時以上之「南瀛陣頭運動體驗課程」。
4. **分區研習與全民體驗工作坊**：於宮廟、社區、學校或香科場域辦理20場以上全民體驗活動。
5. **香科／廟會場域成果展演**：依團隊所屬廟會或地方慶典時程，以精華展演、示範教學或互動導覽方式呈現。
6. **期末品牌成果發表與影像典藏**：辦理成果發表、發布成果影片、主視覺、成果手冊、KPI成果與影音資料庫。

### 7.2 分散式文化祭模式

本節分析本計畫與傳統大型集中式活動不同之處。其特色在於：

1. 活動回到香科、廟會與社區場域。
2. 不強求所有陣頭一次性集中動員。
3. 以子計畫申請制提高團隊自主性。
4. 以精華展演降低成本與安全壓力。
5. 以影像典藏累積長期品牌資產。

### 7.3 主題運動推廣課程設計

本節可分析主題課程如何將陣頭技藝轉化為可教學、可練習、可安全參與的全民運動課程，包括基本步法、節奏律動、龍獅協力、車鼓跳鼓與安全實作。

## 第八章　成效評估、風險管理與永續機制

### 8.1 預期效益分析

本計畫預期效益可整理為六項：

1. 建立臺南地方運動品牌。
2. 提升規律運動人口與參與機會。
3. 活化宮廟與社區場域。
4. 促進文化傳承與青年參與。
5. 創造數位保存資產。
6. 降低大型活動風險。

### 8.2 KPI與活動成效評估

可依申辦計畫規定，分析以下指標：

1. 參與人數達成率。
2. 活動場次執行率。
3. 活動滿意度。
4. 問卷回收率。
5. 媒體與社群宣傳效益。
6. 成果填報率。
7. 安全管理執行情形。
8. 計畫核實性。

### 8.3 安全與風險治理

本計畫風險治理可從以下面向分析：

1. 分區、分期、分眾辦理以降低人流風險。
2. 精華展演替代完整大型陣頭動員。
3. 保險規劃與緊急應變機制。
4. 天候與突發事件因應。
5. 課程出席率與活動核實性管理。
6. 受訪者、學員與參與者資料保護。

### 8.4 永續與品牌延續

本計畫透過主視覺、社群標籤、影音資料庫、成果手冊、年度延續機制與跨域合作網絡，形成可累積之地方運動品牌。此部分可作為後續政策建議之核心。

---

## 第九章　討論

### 9.1 陣頭運動化的可能與限制

本節討論將陣頭轉化為全民運動時，如何避免過度去脈絡化與表演化，並維持其信仰、地方、師承與社群脈絡。

### 9.2 地方文化品牌化的治理意義

探討「南瀛香科陣頭運動文化祭」如何使臺南從活動執行者轉向品牌經營者，並建立具辨識度、可傳承與可累積的城市運動文化。

### 9.3 民俗體育與全民運動政策之銜接

本節討論民俗體育如何進入全民運動政策體系，並提出「民俗體育不只是文化保存，也可成為規律運動人口、社區參與與地方治理的媒介」。

### 9.4 從臺南案例到全國推廣模式

討論此模式是否可作為其他縣市推動地方特色全民運動活動之參考，例如客家庄、原住民族傳統身體活動、農村節慶運動、海洋文化運動等。

---

## 第十章　結論與建議

### 10.1 研究結論

1. 臺南香科與陣頭文化具備轉化為地方特色全民運動品牌的高度潛力。
2. 「陣頭運動」概念可擴展傳統武陣研究範圍，使其涵蓋武藝型、龍獅鼓樂型與民俗律動型身體文化。
3. 分散式文化祭模式有助於兼顧文化真實性、全民參與、經費合理性與風險控管。
4. 數位影像保存與品牌識別可使一次性活動轉化為長期文化資產。
5. 此案可作為民俗體育政策化、品牌化與全民化之重要案例。

### 10.2 實務建議

1. 建議建立臺南陣頭運動資源資料庫。
2. 建議每年依香科週期滾動辦理地方特色陣頭運動活動。
3. 建議建立陣頭運動課程模組與安全教學指引。
4. 建議強化宮廟、學校、社區、武館與地方政府之合作機制。
5. 建議將影像典藏、教學短片與社群傳播納入年度成果指標。

### 10.3 後續研究建議

1. 進行實際活動田野觀察與參與者訪談。
2. 比較不同香科場域中陣頭運動的身體技藝與社群功能。
3. 分析青年、親子與銀髮族對陣頭運動體驗之接受度。
4. 建立民俗體育品牌化與地方治理之理論模型。
5. 比較臺南陣頭運動與高雄內門宋江陣、客家獅、馬來西亞龍獅運動之異同。
